This is a demo of Google Maps API. When first entering the page, it asks to allow use of your location, if allowed, it shows a map of your current location, if declined it defaults to a Milwaukee map. There are custom icons for several of the different location "types" and also a backup default icon if the type does not match the handful I set up. When clicking an icon, it pulls in information and user reviews for the location. On mobile devices the reviews can be swiped left and right. Also shows an animated loader when loading new maps.
